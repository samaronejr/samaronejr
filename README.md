## Research focus

I'm developing my path in AI by studying and building ML systems/pipelines that are easier to reproduce, evaluate, and trust. My current interests include:

- model robustness and reliability in real-world workflows;
- reproducible machine learning experiments and pipelines;
- practical deep learning and classical ML for engineering problems;
- clean, scriptable tooling for data, training, and evaluation.

## Current work

- Strengthening my foundations as an **MSc student in AI**.
- Applying an **Electrical Engineer** background to machine learning systems.
- Building and refining ML pipelines with attention to robustness, reproducibility, and reliability.
